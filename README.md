Watch crypto-currency addresses.

## Contributing

Install dependencies with [`yarn`](https://yarnpkg.com/lang/en/).

```
yarn install
```

Watch for changes and test/build with

```
yarn start
```

Add tests for changes before pushing/opening a PR.
